The word game(Word_raider)
the rules are very simple. the word game will prompt the user to guess a five letter word. 
As the user submits there word, some feedback will be provided as to whether the letters within their guess are in the word they guess
As then game will provide somefeedback as to the whether the letters within their guess are in the word to guess. if the player guesses the correct letter in the correct postion. it will be filled in the console. 
if they guess a correct letter that belongs in the word, but it is in the wrong position, that letter will be added to a list of misplaced letter and an underscore will be shown in that position on the console
if they guess an incorrect letter that doesnot belong in the word, that letter will also be added to a list of incorrect letters and an underscore will be shown in that position on the console. the user will have maximum 5 tries
